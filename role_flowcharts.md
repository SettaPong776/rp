# Flowchart วิธีการใช้งานระบบแจ้งซ่อมออนไลน์ แยกตาม Role

---

## 1. ผู้ใช้งานทั่วไป (User)

```mermaid
flowchart TD
    A["เข้าสู่ระบบ"] --> B["หน้าหลัก (Dashboard)"]
    B --> C["แจ้งซ่อมใหม่"]
    B --> D["รายการแจ้งซ่อมของฉัน"]
    B --> E["ข้อมูลส่วนตัว"]

    C --> C1["กรอกรายละเอียด\nชื่อเรื่อง / หมวดหมู่ / สถานที่"]
    C1 --> C2["แนบรูปภาพ (ถ้ามี)"]
    C2 --> C3["ส่งแบบฟอร์ม"]
    C3 --> C4["รอดำเนินการ"]

    D --> D1["ดูสถานะงานซ่อม"]
    D1 --> D2{"สถานะ?"}
    D2 -->|"รอดำเนินการ"| D3["รอแอดมินรับงาน"]
    D2 -->|"กำลังดำเนินการ"| D4["รอช่างซ่อมเสร็จ"]
    D2 -->|"เสร็จสิ้น"| D5["ดูผลการซ่อม"]
    D1 --> D6["พิมพ์รายงาน (PDF)"]

    E --> E1["แก้ไขข้อมูลส่วนตัว"]
    E --> E2["เปลี่ยนรหัสผ่าน"]

    B --> F["ออกจากระบบ"]
```

## 2. ผู้ดูแลระบบ (Admin)

```mermaid
flowchart TD
    A["เข้าสู่ระบบ"] --> B["แดชบอร์ด (Admin Dashboard)"]

    B --> C["รายการแจ้งซ่อม"]

    B --> E["ผู้ใช้งาน"]

    C3 --> C5["เพิ่มหมายเหตุ / แนบรูป"]
    C3 --> C6["พิมพ์รายงาน (PDF)"]

    D --> D1["เพิ่มหมวดหมู่ใหม่"]
    D --> D2["แก้ไขหมวดหมู่"]
    D --> D3["ลบหมวดหมู่"]

    E --> E1["ดูรายชื่อผู้ใช้"]
    E --> E2["เพิ่มผู้ใช้ใหม่"]
    E --> E3["แก้ไขข้อมูลผู้ใช้"]
    E --> E4["รีเซ็ตรหัสผ่าน"]
    E --> E5["ลบผู้ใช้"]

    F --> F1["ดูสถิติภาพรวม"]

    G --> G1["ตั้งค่าชื่อระบบ"]
    G --> G2["ตั้งค่า Telegram แจ้งเตือน"]

    B --> H["ออกจากระบบ"]
```

## 3. งานอาคาร (Building Staff)

```mermaid
flowchart TD
    A["เข้าสู่ระบบ"] --> B["แดชบอร์ด (Admin Dashboard)"]

    B --> C["รายการแจ้งซ่อม"]
    B --> D["หมวดหมู่"]
    B --> E["ผู้ใช้งาน"]
    B --> F["รายงาน"]

    C --> C1["ดูรายการแจ้งซ่อมทั้งหมด"]
    C1 --> C2["กรองตามสถานะ / หมวดหมู่"]
    C1 --> C3["ดูรายละเอียดงาน"]
    C3 --> C4["อัปเดตสถานะ\nรับงาน → กำลังดำเนินการ → เสร็จสิ้น"]
    C3 --> C5["เพิ่มหมายเหตุ / แนบรูป"]
    C3 --> C6["พิมพ์รายงาน (PDF)"]

    D --> D1["เพิ่มหมวดหมู่ใหม่"]
    D --> D2["แก้ไขหมวดหมู่"]
    D --> D3["ลบหมวดหมู่"]

    E --> E1["ดูรายชื่อผู้ใช้เท่านั้น"]
    E -.->|"❌ ไม่สามารถ"| E2["เพิ่ม / แก้ไข / ลบผู้ใช้"]
    E -.->|"❌ ไม่สามารถ"| E3["รีเซ็ตรหัสผ่าน"]

    F --> F1["ดูสถิติภาพรวม"]
    F --> F2["กรองตามช่วงเวลา"]
    F --> F3["ส่งออกรายงาน"]

    B -.->|"❌ ไม่มีเมนู"| G["ตั้งค่า"]

    B --> H["ออกจากระบบ"]
```
